关于卡常

1. i++ vs ++i

A. for (register int i=0;i<1000000000;i++); 3次平均用时：0.330s

B. for (register int i=0;i<1000000000;++i); 3次平均用时：0.331s

 

2. printf vs cout vs 快速输出

A. for (register int i=0;i<1000000;i++) printf("%d\n",123456789); 3次平均用时：3.533s

B. for (register int i=0;i<1000000;i++) cout<<123456789<<endl; 3次平均用时：7.802s

C. for (register int i=0;i<1000000;i++) print(123456789),putchar(10); 3次平均用时：0.423s

 

3. register

A. 1.A 不加 register 3次平均用时：2.507s

B. 1.A 加上 register 3次平均用时：0.330s

 

4. n*32+6 vs n<<5|6

A. for (register int i=0;i<1000000000;i++) n=n*32+6; 3次平均用时：2.363s

B. for (register int i=0;i<1000000000;i++) n=n<<5|6; 3次平均用时：2.342s
