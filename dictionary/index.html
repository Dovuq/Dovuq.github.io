This is Dovuq's Dictionary.

Tips:

Verbs has different tences. There are Normeno, Fixano, Pastino, Omegano, and Futuro. The verb will write in Normeno. And other tences will write after it. For example: sieque (siequan, sieque, sieque, siequ) Some verbs doesn't have some tences. They will be written /

Noun's plural: usually -(e)s

Adjective's comparative: usually -ji

Adjective's superlative: usually -jest
